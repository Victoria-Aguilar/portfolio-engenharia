# Análise de Indicadores de Manutenção

Projeto com foco na análise de indicadores de confiabilidade e manutenção, como MTBF, MTTR e Downtime, amplamente utilizados na indústria para apoiar decisões estratégicas e operacionais.

## Indicadores analisados

- **MTBF (Mean Time Between Failures)**  
  Mede o tempo médio entre falhas de um equipamento.

- **MTTR (Mean Time To Repair)**  
  Mede o tempo médio necessário para reparo após uma falha.

- **Downtime**  
  Representa o tempo total em que o equipamento permanece indisponível.

## Objetivo do projeto

Demonstrar, de forma prática, como indicadores de manutenção podem ser utilizados para identificar falhas recorrentes, apoiar planos de manutenção preventiva e melhorar a eficiência operacional.

## Ferramentas utilizadas

- Power BI  
- Excel  
- Análise de dados

## Observação

Os dados utilizados neste projeto são fictícios e têm finalidade exclusivamente acadêmica.
